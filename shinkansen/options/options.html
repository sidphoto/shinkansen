Gemini API

模型計價（USD）

用於計算翻譯成本，單位為「每 1M tokens 的美金價格」。切換模型時會自動帶入參考價，你也可以手動修改。請至 Gemini 官方定價頁 查詢最新費率

效能與配額（並行翻譯）

根據你的 Gemini API 付費層級，自動控制翻譯速度，避免超出 Google 的使用量限制

LLM 參數微調

Gemini 2.5+ 系列是 thinking model，開啟後模型會先內部推理再翻譯，可能提升品質但會增加延遲與 token 消耗。預設關閉，因為思考 token 計入 Max Output Tokens 額度，可能導致多段翻譯被截斷。

術語表一致化

長文翻譯時，先擷取全文專有名詞建立術語對照表，確保分批翻譯的名詞譯名一致。短文（1 批以內）自動跳過。
